Stuff I've been doing recently

For the past 6-8 months, in addition to a site-wide upgrade, I've been working hard on a new system for CyberGamer. This new system is inspired by facebook and has the purpose of connecting friends in real-time when they're online. The system features a multi-purpose toolbar, but most importantly a friends list and an activity list.

Using Node.js - which is a platform built on Google's V8 javascript engine - I'm able to keep a constant connection open with all users on CyberGamer through longpolling, and am able to push data to specific users as events happen. This means users are notified when their friends come online or go offline, they can send real-time instant messages to their friends with negligible latency and can see what their friends are up to through the activity listing.
